# 🎥 YouTube Clone with API 

## 📌 Overview
A modern YouTube Clone application that replicates core functionalities of a video streaming platform. It allows users to browse, search, and watch videos using real-time data fetched from YouTube APIs, delivering a seamless and dynamic user experience.

---

## 🛠️ Tech Stack
- React.js
- Redux Toolkit
- Tailwind CSS
- JavaScript (ES6+)
- YouTube Data API / Rapid API

---

## ✨ Features
- 🎥 Video browsing with real-time API data
- 🔍 Search functionality with dynamic results
- ▶️ Video playback with detailed view page
- 📂 Category-based video filtering
- ⚡ Optimized API calls with debouncing & caching
- 🔄 Global state management using Redux Toolkit
- 📱 Fully responsive design (mobile/tablet/desktop)
- 🌙 Dark mode support
